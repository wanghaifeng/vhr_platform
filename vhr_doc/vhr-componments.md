## 5. 数据流组件图

### 5.1 请求处理流程

```
用户请求
    │
    ↓
Vue.js 前端
    │ (Axios HTTP请求)
    ↓
Nginx 反向代理
    │ (请求转发)
    ↓
Spring Security 过滤器链
    │ (认证/授权)
    ↓
Controller 控制器
    │ (请求处理)
    ↓
Service 业务层
    │ (业务逻辑)
    ↓
Mapper 数据访问层
    │ (MyBatis SQL)
    ↓
MySQL 数据库
    │ (返回结果)
    ↓
Service 业务层
    │ (数据处理)
    ↓
Controller 控制器
    │ (响应封装)
    ↓
前端视图渲染
```

### 5.2 邮件发送流程

```
员工入职操作
    │
    ↓
EmployeeService
    │ (保存员工信息)
    ↓
RabbitMQ 消息队列
    │ (异步消息)
    ↓
mailserver 邮件服务
    │ (消息监听)
    ↓
Thymeleaf 模板渲染
    │ (生成邮件内容)
    ↓
SMTP 邮件服务器
    │ (发送邮件)
    ↓
员工邮箱
```

### 5.3 WebSocket 通信流程

```
用户登录
    │
    ↓
建立 WebSocket 连接
    │ (SockJS + STOMP)
    ↓
WsController 处理连接
    │ (会话管理)
    ↓
Redis 存储在线状态
    │
    ↓
聊天消息广播
    │ (消息分发)
    ↓
在线用户接收消息
```

## 8. 组件交互关系总结

```
┌──────────────────────────────────────────────────────────┐
│                     vhr 系统组件交互图                     │
└──────────────────────────────────────────────────────────┘
                           │
         ┌─────────────────┼─────────────────┐
         │                 │                 │
    前端组件           后端组件          中间件组件
         │                 │                 │
         │                 │                 │
    ┌────┴────┐       ┌────┴────┐       ┌────┴────┐
    │ Vue.js │       │ Spring  │       │  MySQL  │
    │  SPA   │←HTTP─→│  Boot   │←SQL──→│ Database│
    └────────┘       └────┬────┘       └─────────┘
                          │
                   ┌──────┼──────┐
                   │      │      │
              ┌────┴┐ ┌──┴───┐ ┌┴─────┐
              │Redis│ │RabbitMQ│ │FastDFS│
              └─────┘ └───────┘ └──────┘
```

**关键交互点：**

1. **前端 ↔ 后端**：通过 Axios 发送 HTTP/HTTPS 请求，Spring Boot 接收并处理
2. **后端 ↔ 数据库**：通过 MyBatis 进行 SQL 操作，Druid 管理连接池
3. **后端 ↔ Redis**：缓存菜单数据、存储 Session、在线用户状态
4. **后端 ↔ RabbitMQ**：异步发送邮件消息、解耦业务逻辑
5. **后端 ↔ FastDFS**：存储员工头像、文档附件
6. **前端 ↔ WebSocket**：实现在线聊天功能
7. **安全交互**：Spring Security 拦截所有请求，进行认证和授权检查
